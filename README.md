# FinEd Landing Page

Financial Education App for Teens - Landing Page

## Overview

This is the marketing landing page for FinEd, a gamified financial education app designed for teens. Built with Next.js 14+ and Tailwind CSS.

## Design System

- **Primary Color**: Blue #3A7AFE
- **Secondary Color**: Green #2ECC71
- **Accent Color**: Yellow #F1C40F
- **Typography**: Poppins font family
- **Border Radius**: 16px

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
